# RFID Smart Parking System (8051)

## Overview
This project implements a smart parking system using the AT89C51 microcontroller.

## Features
- Dynamic slot allocation (5 slots)
- Entry and exit tracking
- Parking full detection
- LCD + Virtual Terminal output

## Technologies
- Embedded C (Keil)
- Proteus Simulation
- 8051 Microcontroller
